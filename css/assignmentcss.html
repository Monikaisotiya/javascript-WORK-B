By keeping related properties together, it becomes easier to locate, modify, and update specific styles. This separation also facilitates collaboration between designers and developers, as they can work on different aspects of the design without conflicting or duplicating each other's efforts. (15) What are the advantages and disadvantages of using external style sheets? ANS: #Advantages of using external style sheets: 1> Separation of concerns: By using external style sheets, you can separate the presentation (CSS) from the structure (HTML) of your web page. This separation improves the maintainability of your code, as changes to the styling can be made in one central location, making it easier to update and modify the appearance of multiple web pages simultaneously. 2>Reusability: External style sheets allow you to define a set of styles that can be reused across multiple web pages. This promotes consistency in the design and ensures that the same styles are applied consistently throughout your website. However, this disadvantage is often mitigated by browser caching, as subsequent requests can be served from the cache. 2> Dependency on external resources: External style sheets are dependent on the availability and proper linking of the CSS file. If the CSS file fails to load or is not accessible, it can result in unstyled web pages or broken layouts. This dependency on an external resource introduces a potential point of failure. (16) What is the meaning of the CSS selector? ANS: In CSS (Cascading Style Sheets), a selector is a pattern used to select and target specific HTML elements on a web page. The selector determines which elements the CSS rules should be applied to. It is used to define the styling and presentation of those selected elements. Selectors can be based on various criteria, such as the element type, class, ID, attributes, or relationship to other elements. Here are some common CSS selectors and their meanings: 1> Element Selector: It selects all instances of a specific HTML element. For example, the selector "p" targets all

elements on the page. 2>Class Selector: It selects elements with a specific class attribute value. The selector ".classname" targets all elements with the specified class. For example, ".header" targets all elements with the class "header". 3>ID Selector: It selects a single element with a specific ID attribute value. The selector "#idname" targets the element with the specified ID. For example, "#logo" targets the element with the ID "logo". 4>Attribute Selector: It selects elements based on their attribute values. For example, "[type='text']" targets all elements with the attribute "type" set to "text". 5>Descendant Selector: It selects elements that are descendants of another element. For example, "div p" targets all

elements that are descendants of a

These are just a few examples of CSS selectors. CSS provides a wide range of selectors that allow for precise targeting and styling of HTML elements on a web page. (17) What are the media types allowed by CSS? ANS: CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation and formatting of a document written in HTML (Hypertext Markup Language) or XML (eXtensible Markup Language). CSS is primarily used to control the visual aspects of a web page or document. In CSS, there are several media types that allow you to specify different styles for different output devices or media. The media types supported by CSS are as follows: 1>all: Applies to all devices and media types. 2>print: Intended for print preview and printing. 3>screen: Intended for computer screens, tablets, smartphones, and other similar devices. 4>speech: Intended for screen readers and other speech synthesizers. These media types are used in CSS media queries, which allow you to specify different styles for different media types. For example, you can define styles specifically for printing by using the @media print rule, or you can define styles specifically for screens by using the @media screen rule.
